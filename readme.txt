Create peculiar Buttons with one simple Shortcode, suitable for beginners and pros

== Description ==
Create peculiar Buttons in 12 differnet Colors or whole hexadecimal-Code, using own labelling and individual target-URLs. A special settings-page explains all functions.

== Installation ==
1. Upload folder `mcjh-buttons` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Ready. Now create your first button with [createButton]! See more Information in the Settings-Page
